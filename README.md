# shadcn-starter

A modern React + Vite + Tailwind CSS + shadcn/ui starter template by **Gavin Pereira**.

> ⚡ Get started building elegant and fast React apps with built-in routing and pre-styled UI components.

---

## ✨ Features

- ⚛️ **React** (with TypeScript)
- ⚡ **Vite** — fast dev server and optimized builds
- 🎨 **Tailwind CSS v4**
- 🧩 **shadcn/ui** components pre-installed
- 🧭 **React Router v6** for client-side routing
- 🧹 Pre-configured with **Prettier**
- ✅ Example layout, pages, and component structure

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/pexeixv/shadcn-starter
```

### 2. Install dependencies (using pnpm)

```bash
pnpm install
```

### 3. Start the development server

```bash
pnpm run dev
```

### 4. Build for production

```bash
pnpm run build
```

---

## 📁 Folder Structure

```
src/
├── components/         # Reusable UI components (shadcn/ui)
├── layout/             # Base layout, header, and footer
├── pages/              # App routes
├── App.tsx             # Routes setup
└── main.tsx            # Entry point
```

---

## 🛠 Customization Tips

- Add new pages in `src/pages/` and routes in `App.tsx`.
- Use `src/components/ui/` for ready-to-use shadcn components.
- Customize Tailwind in `tailwind.config.ts`.
- Modify layout via `BaseLayout.tsx`.
